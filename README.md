This Fish shell greeting shows an image when initilizing the shell.





Dependencies:














- chafa










  chafa is the program used in order to display the image on your terminal.





















- A terminal with image support, such as Konsole or Ghostty.









Installation






- Edit the file and add your images folder.





- Copy or move the file into ~/.config/fish/functions




- Initialize the shell to see a random image.










Have fun!
